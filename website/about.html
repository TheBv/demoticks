About

Demoticks.tf is a utility that allows the user to find relevant events in their recent Team Fortress 2 matches uploaded to logs.tf. It is mostly meant to simplify the workflow of finding points of interest in Team Fortress 2 demo files.

API

Demoticks.tf provides a GraphQL API with three major functions:

  • AddLog

    Allows adding a logfile by id to the demoticks.tf database.

  • SearchPlayer

    Enables the search through any tf2 player by name or Steam64ID
    This only works for people registerd to either etf2l.org, ozfortress.com or logs.tf the name returned also derives in that order.
    The RGL API is currently not used since it's slow and also being worked on.

  • SearchEvents

    Enables the search through the database of events by various search terms.

The URL and schema can be found here: Schema | demoticks.tf/graphQL
